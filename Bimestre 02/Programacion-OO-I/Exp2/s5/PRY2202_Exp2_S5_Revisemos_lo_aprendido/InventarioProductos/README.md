# Proyecto de Camila: Inventario de Productos

Camila está construyendo un sistema para cargar productos desde un archivo de texto. Quiere asegurarse de que su programa no falle si encuentra errores o datos incompletos.

## Qué observar en este proyecto:
- Clase `Producto.java` en el paquete `model`, que representa un producto
- Clase `GestorProductos.java` en el paquete `data`, con el método `cargarDesdeArchivo()` que incluye validaciones
- Uso de bloques `try-catch` para manejar errores de carga
- Estructura del proyecto organizada en paquetes funcionales
