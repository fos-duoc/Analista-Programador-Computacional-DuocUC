# Bimestre 12 - Desarrollo Fullstack II & Cloud Computing

## Ingeniería en Desarrollo de Software - DuocUC

Profundización en desarrollo fullstack y fundamentos de computación en la nube.

---

## Asignaturas

### 1. Desarrollo Fullstack II
Desarrollo avanzado de aplicaciones web con arquitecturas modernas.

**Competencias:**
- Arquitectura de microservicios
- WebSockets y comunicación en tiempo real
- Server-Side Rendering (SSR)
- Progressive Web Apps (PWA)
- Testing E2E

**Tecnologías:**
- Next.js / Nuxt.js
- GraphQL
- Socket.io
- Cypress / Playwright
- Docker containers

### 2. Cloud Computing
Fundamentos y prácticas de computación en la nube.

**Competencias:**
- Servicios cloud (IaaS, PaaS, SaaS)
- Infraestructura como código
- Contenedores y orquestación
- Servicios serverless
- Seguridad en la nube

**Tecnologías:**
- AWS / Azure / GCP
- Docker
- Terraform
- Lambda / Cloud Functions
- S3 / Blob Storage
