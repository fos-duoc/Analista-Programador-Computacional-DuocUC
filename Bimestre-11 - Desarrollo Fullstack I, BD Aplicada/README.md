# Bimestre 11 - Desarrollo Fullstack I & Base de Datos Aplicada

## Ingeniería en Desarrollo de Software - DuocUC

Este bimestre marca el inicio de la carrera de **Ingeniería en Desarrollo de Software**, continuación del programa de Analista Programador Computacional.

---

## Asignaturas

### 1. Desarrollo Fullstack I
Introducción al desarrollo de aplicaciones web completas, integrando frontend y backend.

**Competencias:**
- Arquitectura de aplicaciones web modernas
- Integración de APIs RESTful
- Frameworks frontend (React/Angular/Vue)
- Frameworks backend (Spring Boot/Node.js)
- Autenticación y autorización

**Tecnologías:**
- React / Angular / Vue.js
- Node.js / Spring Boot
- TypeScript
- RESTful APIs
- JWT Authentication

### 2. Base de Datos Aplicada
Diseño e implementación avanzada de bases de datos para aplicaciones empresariales.

**Competencias:**
- Modelamiento de datos avanzado
- Optimización de consultas
- Procedimientos almacenados
- Triggers y funciones
- Replicación y alta disponibilidad

**Tecnologías:**
- PostgreSQL / MySQL
- MongoDB (NoSQL)
- Redis (Cache)
- Indexación y Query Optimization
