@RestController
@RequestMapping("/api/usuarios")
public class UsuarioController {

    @Autowired
    private UsuarioService usuarioService;

    @GetMapping("/{id}")
    public ResponseEntity<Usuario> obtenerUsuario(@PathVariable Long id) {
        Usuario usuario = usuarioService.buscarPorId(id);
        if (usuario == null) {
            return ResponseEntity.notFound().build();
        }
        return ResponseEntity.ok(usuario);
    }

    @PostMapping
    public ResponseEntity<Usuario> crearUsuario(@Valid @RequestBody Usuario usuario) {
        Usuario nuevo = usuarioService.guardar(usuario);
        return ResponseEntity.status(HttpStatus.CREATED).body(nuevo);
    }
}

Query con JOIN y agregación

Consulta compleja con múltiples JOINs

query_con_join_y_agregación.sql
14 líneas
1234567891011121314
SELECT
    d.nombre AS departamento,
    COUNT(e.id) AS total_empleados,
    AVG(e.salario) AS salario_promedio,
    MAX(e.salario) AS salario_maximo,
    MIN(e.fecha_contratacion) AS primera_contratacion
FROM departamentos d
LEFT JOIN empleados e ON d.id = e.departamento_id
WHERE e.activo = true
    AND e.fecha_contratacion >= '2023-01-01'
GROUP BY d.id, d.nombre
HAVING COUNT(e.id) > 5
ORDER BY total_empleados DESC
LIMIT 10;

async function fetchUsuarios() {
    try {
        const response = await fetch('https://api.ejemplo.com/usuarios', {
            method: 'GET',
            headers: {
                'Content-Type': 'application/json',
                'Authorization': `Bearer ${token}`
            }
        });

        if (!response.ok) {
            throw new Error(`HTTP error! status: ${response.status}`);
        }

        const data = await response.json();
        return data;

    } catch (error) {
        console.error('Error fetching usuarios:', error);
        throw error;
    }
}

        body: JSON.stringify(usuario)
    });
    return response.json();
}

React Hook personalizado

Custom hook para manejo de datos

import { useState, useEffect } from 'react';

function useFetch(url) {
    const [data, setData] = useState(null);
    const [loading, setLoading] = useState(true);
    const [error, setError] = useState(null);

    useEffect(() => {
        const abortController = new AbortController();

        async function fetchData() {
            try {
                setLoading(true);
                const response = await fetch(url, {
                    signal: abortController.signal
                });
                const json = await response.json();
                setData(json);
                setError(null);
            } catch (err) {
                if (err.name !== 'AbortError') {
                    setError(err.message);
                }
            } finally {
                setLoading(false);
            }
        }

        fetchData();
        return () => abortController.abort();
    }, [url]);

    return { data, loading, error };
}

Preguntas Frecuentes

Respuestas a las dudas más comunes sobre el programa

¿Cuánto dura la carrera completa?

Analista Programador Computacional: 10 bimestres (20 meses)

+ Ingeniería en Desarrollo de Software: 7 bimestres adicionales (14 meses)

Total con ambos títulos: 34 meses de estudio

¿Es 100% online?

Sí, el programa es completamente online con modalidad asíncrona y síncrona.

  • Clases grabadas disponibles 24/7
  • Sesiones en vivo opcionales
  • Evaluaciones online
  • Proyecto de título remoto
¿Qué lenguajes de programación se aprenden?

Los principales lenguajes del programa son:

  • Python - Fundamentos y análisis de datos
  • Java - Programación orientada a objetos
  • JavaScript/TypeScript - Desarrollo web
  • SQL - Bases de datos

Además se trabaja con frameworks como Spring Boot, React, Angular y más.

¿Cuál es el campo laboral?

Los egresados pueden trabajar como:

  • Desarrollador Full Stack
  • Desarrollador Backend / Frontend
  • Ingeniero DevOps
  • Desarrollador Mobile
  • Data Engineer
  • Arquitecto de Software
¿Qué herramientas necesito?

Hardware mínimo:

  • Computador con 8GB RAM (16GB recomendado)
  • Procesador i5 o equivalente
  • 50GB de espacio libre

Software (gratuito):

  • VS Code, IntelliJ IDEA Community
  • Git, Docker Desktop
  • Node.js, Python, Java JDK
¿Qué títulos se obtienen?

Al completar los 10 bimestres:

Técnico de Nivel Superior en Analista Programador Computacional

Al completar los 17 bimestres:

Ingeniero(a) en Desarrollo de Software

Ambos títulos están acreditados y reconocidos por el Ministerio de Educación de Chile.

¿Hay trabajo en equipo?

Sí, el programa incluye:

  • Proyectos grupales cada bimestre
  • Metodologías ágiles (Scrum)
  • Uso de GitHub para colaboración
  • Code reviews entre pares
  • Proyecto de título en equipo
¿Hay prácticas profesionales?

El programa incluye:

  • Proyectos con casos reales de la industria
  • Talleres con profesionales activos
  • Desarrollo de portafolio profesional
  • Conexión con empresas para prácticas (opcional)
