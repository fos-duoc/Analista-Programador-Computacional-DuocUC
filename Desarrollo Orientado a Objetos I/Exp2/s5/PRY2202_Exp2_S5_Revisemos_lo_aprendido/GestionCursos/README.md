# Proyecto de Pedro: Gestión de Cursos

Pedro ya desarrolló su aplicación Java para gestionar cursos y ahora necesita empaquetarla en un archivo `.jar` para compartirla fácilmente.

## Qué observar en este proyecto:
- Archivo `pom.xml` con la configuración Maven
- Paquetes `model`, `data` y `ui` organizados por responsabilidad
- Archivo `.jar` generado automáticamente en la carpeta `target`
- Clase `Main.java` como punto de entrada de la aplicación
