# Proyecto de Ignacio: Librería Modular

Ignacio está desarrollando un sistema que le permita registrar libros a partir de un archivo CSV, organizando su proyecto de forma clara y funcional.

## Qué observar en este proyecto:
- Clase `Libro.java` en el paquete `model`, que representa cada libro
- Clase `GestorLibros.java` en el paquete `data`, que gestiona la lista de libros usando `ArrayList`
- Clase `Main.java` en el paquete `ui`, que inicia el programa
- Estructura de paquetes: `model`, `data` y `ui`
- Archivo `libros.csv` desde donde se cargan los datos
