### 1. Desarrollo Fullstack III
Desarrollo de aplicaciones empresariales escalables.

- Event-driven architecture
- Message queues

**Tecnologías:**
- Kafka / RabbitMQ

- Kubernetes

Cultura y prácticas de integración y entrega continua.

**Competencias:**
- CI/CD pipelines
- Infrastructure as Code
- Monitoreo y observabilidad
- GitOps
- Site Reliability Engineering

**Tecnologías:**
- GitHub Actions / GitLab CI
- Jenkins
- Prometheus / Grafana
- ArgoCD
- Ansible
